О приложении
Загрузка текстов
Данный раздел предназначен для загрузки текстов на английском языке и работы с ними.
В левой части имеются поля для вставки текста и его наименования. При нажатии кнопки "Загрузить статью" он автоматически сохраняется и он доступен для просмотра во вкладке "Статьи".
После загрузки текст доступен для работы. Все слова, уже имеющиеся в словаре, обладают всплывающими подсказками с переводом. А новые слова отмечены красным цветом. Их список также приведен ниже для ручного перевода. Каждое слово подлежит перевести, после чего следует подтвердить окончание работы нажатием кнопки "Сохранить в словарь". Если оставить слово без перевода, то оно не будет добавлено в словарь.
После сохранения слов форма снова доступна для загрузки следующего текста.
Словарь
В словарь попадают слова со своим переводом после загрузки и обработки текста.
Можно начать работу с данным разделом, закачав базовый словарь нажатием соответствующей кнопки. Он содержит 75 общеупотребительных слов английского языка.
Вы также можете напрямую добавлять слова, используя соответствующую кнопку. Слова доступны для редактирования и удаления. Также работает поиск по словам (в английском варианте).
! Предупреждение
Данное приложение работает в тестовом режиме и хранит данные в рамках одной сессии. Опция сохранения словаря в базу данных находится в разработке. Для сохранения результата Вашей работы рекомендуется скачать словарь в файл нажатием соответствующей кнопки.
Статьи
В данном разделе Вы найдете все загруженные тексты, с которыми Вы успешно провели переводческую работу. Для примера в приложении по умолчанию представлено два текста.
